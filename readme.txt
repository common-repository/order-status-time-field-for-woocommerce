Records the time of an order status change as an order meta field for reporting purposes.

== Description ==

This plugin records the date and time of an order's most recent change to a specific status in an order meta field for reporting purposes.

To report on the fields created by this plugin, check out [Export Order Items Pro](https://potentplugins.com/downloads/export-order-items-pro-wordpress-plugin/?utm_source=order-status-time-field-for-woocommerce&utm_medium=link&utm_campaign=wp-plugin-referral).


== Installation ==

1. Click "Plugins" > "Add New" in the WordPress admin menu.
1. Search for "Order Status Time Field for WooCommerce".
1. Click "Install Now".
1. Click "Activate Plugin".

Alternatively, you can manually upload the plugin to your wp-content/plugins directory.
